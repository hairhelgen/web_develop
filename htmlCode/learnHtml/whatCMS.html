Что такое CMS

CMS - "система управления конентом" (движок) - написанная PHP-программистами основа для сайта, с помощью которй вы сможете управлять сайтом (добавлять контент, менять пункты меню и т.п.) не зная HTML и CSS.

Однако, для того чтобы сделать сайт с помощью CMS потребуются услуги и прораммиста, и дизайнера, и верстальщика.

Какие бывают cms

Бывают различные системы управления контентом: для интернет-магазинов, для блогов, для форумов и т.д.

Примеры cms

Примеры популярных CMS: Joomla, WordPress (для блогов), PhpBB (для форумов).

CMS-ки бывают платные и бесплатные.
