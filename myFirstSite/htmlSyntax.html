Здесь располагется конспект по синтаксису

Первое и самое важное это указание на какой версии html ты пишешь:

  • !DOCTYPE html

Далее что ты укажешь это тег самой страницы внутри которого ты и будешь работать:

  • html (/)

Дадим настройки для твоего сайта внутри тега:

  • head (/)

На данный момент я там размещаю кодировку страницы utf-8 (думаю позже я с ней разберусь, сейчас у меня картинка не грузится на сайте) и заголовок сайта (на сколько помню он нужен для SEO)

  • meta
  • title (/)

(/) так я буду обозначать, что требуется закрывающий тег

Ну и наконец наше "тело" страницы, что в ней содержится:

  • body (/)

А вот и основные теги для работы с текстом:

  1. p (/) - абзац
  2. table (/) - таблица
  3. h1...h6 (/) - заголовок
  4. ul, ol, li (/) - список

Тег для работы с сылками:

  • a (/) - открывающий тег для размещения адреса страницы
  • href - тег с указанием адреса страницы, указывается внутри открывающего тега "а"

Тег для работы с изображениями:

  • img
  • src - тег с указанием адреса изображения
